# vd-lan

```
yarn build
node dist/deploy.js --help
```
or

```
yarn start -h
```

```
Options:
      --version  Show version number                                   [boolean]
  -w, --watch    watch mode, only forntend/build.                      [boolean]
  -a, --all      rsync all file to compile.                            [boolean]
  -g, --git      rsync only files from git stash.                      [boolean]
  -u, --usync    Run kantu to do umbraco uSync import.                 [boolean]
  -n, --nuget    Restore nuget packages.                               [boolean]
  -c, --msbuild  Compile with msBuild.                                 [boolean]
  -h, --help     Show help                                             [boolean]
```
